# OpiniQ (오피니큐) - 인사이트 대시보드

## Overview
비정형 텍스트 데이터(댓글, 리뷰 등)를 분석하여 핵심 인사이트를 제공하는 PoC 웹 애플리케이션입니다.
"여론(Opinion)에 지능(IQ)과 단서(Cue)를 더하다"라는 의미의 브랜드명을 사용합니다.

## Project Structure
- `app.py` - Streamlit 메인 애플리케이션
- `.streamlit/config.toml` - Streamlit 서버 설정

## Branding
- **서비스명**: OpiniQ (오피니큐)
- **의미**: Opinion + IQ + Cue
- **로고**: 보라/인디고 그라데이션의 Q자 글자 (심플한 디자인)
- **태그라인**: "여론(Opinion)에 지능(IQ)과 단서(Cue)를 더하다"

## UI/UX Design
- **색상 테마**: 중립 톤 (회색 #374151) + 데이터 시각화 색상 (빨강/파랑/초록)
- **배경색**: #f5f5f5 (라이트 그레이)
- **카드**: 흰색 배경 (result-section)
- **폰트**: Inter (Google Fonts)
- **아이콘**: 일관된 선형 스타일 (stroke-width: 1.5)
- **가독성 최적화**: 큰 헤더, 적절한 여백, 명확한 계층 구조

## Features
1. **데이터 입력**:
   - 4개의 클릭 가능한 버튼: 파일 업로드, 웹사이트, 이미지, 텍스트
   - 버튼 클릭 시 해당 입력 영역 토글
   - 맥락(Context) 추가 기능 (선택 사항)
2. **긴급 이슈 알림**: 신호등(초록/노랑/빨강) 표시로 즉각 대응 필요 이슈 안내
3. **감정 기상도**: 긍정/부정/중립 비율 도넛 차트
4. **주제별 분포**: 주요 토픽 비율 시각화
5. **주제별 상세 분석 (Micro View)**:
   - 클릭 가능한 주제 헤더 버튼
   - 클릭 시 세부 항목 펼침/접기
   - 첫 번째 주제 기본 펼침
   - 진행률 바와 비율 표시
6. **개선 과제**: 우선순위별 상세 설명 포함 액션 아이템
7. **대표 의견**: 긍정/부정 하이라이트
8. **전체 댓글 보기**: 필터링된 댓글 목록

## Profanity Filter (악플 필터링)
- 욕설/비속어를 맥락에 맞게 순화하여 표시
- 예: "ㅅㅂ" → "정말", "지랄" → "난리"
- 수정된 댓글에 "[수정됨]" 배지 표시

## Sidebar Features
- 분석 내역: 최신 순 정렬 (최근 5개 표시)
- "더보기" 버튼으로 추가 내역 확인
- 새로운 분석 버튼
- 데모 데이터 테스트 모드
